## üì¶ Project Structure

The application manages three main components:

- **Inventory**: A list of dictionaries, where each dictionary represents a product with the keys:
  - `name`
  - `price`
  - `quantity`

- **Clients**: A dictionary that stores client information and their purchase history.

- **Total Sales**: A variable that keeps track of the total sales of the store.
